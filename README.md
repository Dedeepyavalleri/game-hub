# 🎮 Game Hub
A Python-based desktop application that brings multiple mini-games together in one place using Tkinter.
## Games Included
* Number Guessing Game
* Rock Paper Scissors
* Tic-Tac-Toe
* Word Guessing Game
* Hangman
## Technologies Used
* Python
* Tkinter
## Features
* Simple and user-friendly GUI
* Multiple games in a single application
* Interactive gameplay
* Lightweight desktop application
## Run Locally
```bash
python main.py
```
